# Evaluation Server

본 프로젝트는 인터뷰 답변을 채점하기 위한 FastAPI 기반 채점 서버입니다.  
사용자 답변이 모범 답변과 **논리적으로 올바른지**,  
**의미적으로 유사한지**,  
**핵심 개념을 포함하고 있는지**를 종합적으로 판단합니다.

본 서버는 Spring Backend와 분리된 독립 서비스로 동작합니다.

---

## 입력 값

채점 서버는 다음 입력을 기준으로 채점을 수행합니다.

- 사용자 답변
- 모범 답변
- 핵심 키워드 목록

---

## 채점 로직 개요

채점은 단일 지표가 아닌, 다음 단계로 구성된 로직을 통해 수행됩니다.

1. 논리 검증 (NLI)
2. 문장 유사도 평가
3. 핵심 키워드 포함 여부 평가
4. 점수 합산 및 최종 판단

---

## 채점 단계

### 1. 논리 검증

- 사용자 답변과 모범 답변을 자연어 추론(NLI) 모델로 비교합니다.
- 결과가 `contradiction`인 경우, 즉시 오답으로 판단합니다.

논리적으로 모순되는 답변은  
이후 단계에서 높은 유사도를 보이더라도 정답으로 처리하지 않습니다.

---

### 2. 문장 유사도 평가 (가중치 0.5)

- 사용자 답변과 모범 답변을 임베딩으로 변환합니다.
- 문장 간 코사인 유사도를 계산하여 점수화합니다.
- 전체 점수의 50%를 반영합니다.

---

### 3. 핵심 키워드 포함 여부 평가 (가중치 0.5)

- 핵심 키워드를 임베딩으로 변환합니다.
- 사용자 답변과의 의미적 유사도를 계산합니다.
- 단순 문자열 비교가 아닌 의미 기반으로 평가합니다.
- 전체 점수의 50%를 반영합니다.

---

### 4. 최종 판단

- 문장 유사도 점수와 키워드 점수를 합산하여 최종 결과를 결정합니다.

---
